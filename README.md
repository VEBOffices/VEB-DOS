# VEB Disk Operating System
A Lightweight Disk Operating System to Rival OpenOS

## About VEB DOS
VEB Disk Operating System is a Lightweight Operating System designed to give developers more freedom over the system, whilst also defaulting the ability to use multiple programs and terminals at once. This DOS is also designed to run with far less memory usage, disk usage, and with lower component overhead.

### Lua Changes
VEB DOS makes a few distinct changes to the Lua Environment OpenOS and ComputerCraft Programmers may be used to, first of all, there is no longer a require, instead, each program is run with a custom wrapped environment of Global, and all the important libraries including the ones part of the OpenComputers Firmware are simply already there, and many libraries which you may be used to no longer exist, or have changed functions, for example, computer.pullSignal() now updates the keyboard before actually responding to a signal.
